# Creating Your Social Media Preview Image (og-image.png)

You need to create an og-image.png file for your portfolio to display properly when shared on social media platforms like Facebook, Twitter, and LinkedIn.

## Specifications

- **Dimensions**: 1200 x 630 pixels
- **Format**: PNG or JPEG
- **File name**: og-image.png
- **Location**: Place in the public folder (/web-dashboard/public/)

## Design Tips

1. **Include your photo**: Since you already have a professional photo (as seen in your favicon), include it in the OG image
2. **Add your name and title**: "Kayes | Full Stack Web Developer"
3. **Use consistent branding**: Match colors with your portfolio website
4. **Keep it simple**: Don't overcrowd with too much information
5. **High contrast**: Ensure text is readable on various backgrounds

## Tools to Create OG Image

1. **Canva** (https://www.canva.com/) - Free and easy to use
2. **Figma** (https://www.figma.com/) - More professional design tool
3. **Adobe Photoshop** - If you have access
4. **Online OG Image Generators**:
   - https://og-image.vercel.app/
   - https://www.opengraph.xyz/
